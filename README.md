#  Notes App - Figma Styled Dashboard

A sleek, responsive, single-page note-taking web application built using Python, Flask, and custom semantic HTML/CSS. The user interface is completely modeled after a custom dark-mode workspace dashboard design originally laid out in Figma.

##  Features
* **Split-Pane Layout:** Left-hand sidebar dedicated to searching, category filtering, and note cards; right-hand viewport dynamically adjusts to display a blank state or an active editor.
* **Dynamic View States:** Uses Flask routing to toggle the main panel seamlessly between a "No note selected" message and a clean "New Note" editor form.
* **Modern Dark-Mode UI:** High-fidelity black and deep gray workspace canvas featuring rounded borders, customized form inputs, and custom tag pills.
* **Structured Data:** Each note entry tracks a customized Title, Category classification (Personal/Work), Timestamp, and Body content.

##  Tech Stack
* **Backend:** Python 3, Flask (Web Framework)
* **Frontend:** HTML5, CSS3 (Flexbox, CSS Grid)
* **Design Reference:** Figma Custom Wireframe
